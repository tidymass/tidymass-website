更新 tidymass

update_tidymass() 函数可以用于更新 tidymass 及其相关的包。

tidymass::update_tidymass(from = "gitlab")

如果 from = "gitlab" 不起作用，您可以更改 from 参数。

  1. “github”

  2. “gitee”

  3. “tidymass.org”

如果所有 from 参数的选项都不起作用，您可以尝试将 fastgit 设置为 TRUE。

tidymass::update_tidymass(from = "gitlab",
                          fastgit = TRUE)
